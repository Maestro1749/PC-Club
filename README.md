# PC CLUB - Backend сервис на Go

REST API сервис для управления компьютерным клубом с поддержкой бронирования ПК и списания средств с баланса пользователя.

## Функционал:
- Регистрация/авторизация пользователей
- Управление компьютерами (добавление/удаление)
- Бронирование компьютера на определенное время
- Проверка доступности компьютера
- Логирование действий системы

## Архитектура:
Проект построен на слоистой архитектуре:
- **handler** - обработка http-запросов
- **service** - бизнес-логика
- **repository** - работа с базой данных

## Технологии:
- **Go**
- **PostgreSQL**
- **Docker**
- **REST API**

## Запуск проекта:
`docker compose up --build`

## Планы:
- Защита от race condition при одновременном бронировании
- Планируется система отмены брони
- Покрытие тестами
- Прочие корректировки

## Статус:
Проект находится в разработке.
