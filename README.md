PC club — Backend сервис на Go

REST API сервис для управления пользовтелями

Функционал:
- Регистрация/авторизация пользователей
- CRUD операции
- Логирование действий
- Работа с БД PostgreSQL

Технологии:
- Go
- POstgreSQL
- Docker
- REST API

Архитектура:
- Разделение бизнес-логики и доступа к данным
- handler/service/repository

Запуск:
- docker compose up
